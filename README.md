# Portfolio de Benjamin Mwaku

Site portfolio personnel de **Benjamin Mwaku**, etudiant a l'Universite Don Bosco de Lubumbashi, centre sur son profil, son parcours, ses passions et ses objectifs dans le web et la cybersecurite.

## Apercu

Le site presente :

- une section d'accueil avec presentation personnelle
- une section **A propos**
- une section **Passions**
- une section **Parcours**
- une section **Technologies**
- une section **Contact** avec liens GitHub, LinkedIn et email

L'interface met l'accent sur :

- un design moderne avec Tailwind CSS
- une navigation responsive avec menu mobile
- des animations d'apparition au scroll
- un effet de texte tape pour la signature principale

## Technologies utilisees

- HTML5
- CSS3
- Tailwind CSS
- JavaScript

## Structure du projet

```text
Myportfolio/
|-- assets/
|   |-- css/
|   `-- img/
|-- src/
|   |-- input.css
|   `-- output.css
|-- index.html
|-- script.js
|-- package.json
`-- README.md
```

## Fonctionnalites

- page one-page simple et responsive
- navigation par ancres
- menu mobile avec ouverture/fermeture en JavaScript
- animation de signature dans le bloc hero
- effets de reveal avec `IntersectionObserver`
- integration d'images personnelles

## Lancer le projet

Comme il s'agit d'un site statique, il suffit d'ouvrir `index.html` dans un navigateur.

Si vous souhaitez regenerer le CSS avec Tailwind :

```bash
npm run build:css
```

Pour surveiller les modifications :

```bash
npm run watch:css
```
